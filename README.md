# 🚀 Monish Gowda M — Personal Portfolio Website

A sleek, responsive, and interactive personal portfolio website built with pure **HTML, CSS, and JavaScript** — no frameworks, no dependencies. Designed with a dark cyberpunk/tech aesthetic featuring glowing cyan accents, smooth animations, and an AI-powered chatbot assistant.

---

## 🌐 Live Preview

> Open `https://my-portfolio-three-mu-32.vercel.app/` directly in any browser — no server required.

---

## ✨ Features

### 🎨 Design & UI
- Dark engineering-themed aesthetic with **cyan glow effects**
- Animated **grid background** with scanline overlay
- Fully **responsive** across all screen sizes (desktop, tablet, mobile)
- Smooth **scroll animations** — elements animate in and out as you scroll up and down
- Hover effects on all cards (projects, skills, education)

### 🧭 Navigation
- Fixed navbar with **active link highlighting** on scroll and click
- **Smooth scroll** to sections with navbar offset correction
- **Mobile hamburger menu** with slide-in drawer, staggered animations, and background blur
- Auto-closes menu and highlights correct link on section entry

### 👤 Hero Section
- Animated **circular photo frame** with spinning conic gradient ring and floating effect
- Badge tag: **AI/ML · Software · Engineer**
- CTA buttons: **Get In Touch** (opens Gmail) and **View Projects** (smooth scrolls)

### 📬 Let's Connect
- Clickable buttons for **Gmail**, **Phone**, **LinkedIn**, and **GitHub**
- All links open in a new tab

### 💼 Work Experience
- Animated **timeline** with glowing dots
- Roles: AI/ML Engineer Intern @ BEEP | Front-End Developer Intern @ Kun Solutions

### 🛠️ Projects
- **Lip-to-Speech Prediction System** — OpenCV + TensorFlow
- **Business Intelligence Dashboards** — Power BI + Tableau
- **TecXCse 2024 Symposium Website** — HTML + MySQL

### 🧠 Technical Skills
- Grouped by category: Languages, Data & BI, Libraries, Developer Tools
- Hover highlight effect on all skill cards

### 🎓 Education
- ACS College Of Engineering, RNS PU College, SJR Kengeri Public School
- Cards with hover glow effect

### 📜 Certificates
- All 4 certificates are **clickable** and open the actual certificate via Google Drive in a new tab:
  - Software Engineering — Forage
  - Generative AI — Intellipath
  - Java Training — Spoken Tutorial
  - UI/UX and AI/ML — Beep

### 🤖 AI Chatbot Assistant
- Floating chat button (bottom-right) with a **live pulse indicator**
- Answers questions about Monish's resume — skills, experience, projects, education, certifications, hobbies, availability
- For out-of-scope questions, provides **direct contact links** (Gmail + Phone)
- **Smooth open/close animations** (slide-up in, slide-down out)
- **Auto-resets** chat history every time it's closed — fresh start each session
- Fully responsive on all screen sizes


> Everything — HTML, CSS, JavaScript, and the embedded photo — is contained in a **single HTML file**. No external dependencies required except Google Fonts (loaded via CDN).
